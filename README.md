# Pizza Sales Analysis

## Project Overview

This project analyzes Pizza Sales data using SQL, Pandas, and Power BI to understand sales performance, customer purchasing patterns, product performance, and time-based sales trends.

## Dataset

The project uses Pizza Sales data containing:

* Orders
* Pizza Details
* Pizza Categories
* Sales Information

## SQL Analysis

Performed business analysis using SQL:

* Total Revenue
* Total Orders
* Average Order Value
* Top Selling Pizzas
* Bottom Selling Pizzas
* Revenue by Category
* Revenue by Pizza Size
* Monthly Sales Trend
* Daily Sales Trend
* Peak Sales Hours

## Pandas Analysis

Performed data cleaning and analysis using Pandas:

* Data Understanding
* Null Value Check
* Date Conversion
* KPI Analysis
* Product Analysis
* Time-based Analysis
* Sales Trend Analysis

## Power BI Dashboard

Created interactive dashboards for business insights:

### Executive & Time Dashboard

* Total Revenue
* Total Orders
* Average Order Value
* Monthly Revenue Trend
* Daily Trend
* Hourly Sales Trend

### Product Dashboard

* Best Selling Pizzas
* Worst Selling Pizzas
* Revenue by Category
* Revenue by Size
* Product Performance Analysis

## Tools Used

* SQL
* Python (Pandas)
* Power BI
* Microsoft Excel

## Key Insights

* Identified top-performing pizzas.
* Analyzed category-wise revenue contribution.
* Measured sales trends by day and hour.
* Found peak sales periods.
* Built interactive dashboards for business decision-making.
